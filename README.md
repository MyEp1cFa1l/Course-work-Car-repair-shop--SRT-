# Course-work-Car-repair-shop--SRT-

Курсовая работа "Автомастерская (СТО)", выполненная в MySQL Workbench
Информационная система учета работы станции технического обслуживания автомобилей

**Используемые технологии**
- MySQL
- MySQL Workbench

**Возможности**

✔ учет клиентов
✔ учет автомобилей
✔ сотрудники
✔ услуги
✔ заказы
✔ отчеты

**Структура БД**
1) ER-диаграмма
2) БД
3) SQL
